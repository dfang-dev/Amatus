### Aira001

## Note
This project is in the alpha stage, very early development. Please let me know any suggestions you may have.

## Reference
This project is heavily inspired by "Neuro-Sama" created by Vedal.
It is also inspired by "Amadeus" from the fiction "Steins;Gate"

## Introduction
Aira001 (Aira for short) is a real time, fully responsive AI Vtuber powered by OpenAI's LLM. She is built with Python and designed to retain memory of conversations which introduces evolving dialogue.
My goal is to make a virtual connection feel "human-like". Through this project I hope to help people overcome their speaking limitations through speaking to Aira, and slowly gain confidence in communicating with others!
